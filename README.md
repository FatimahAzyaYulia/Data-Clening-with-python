# Data Cleaning with Python

## 📌 Project Overview

This project demonstrates the data cleaning process using Python and Pandas. The objective is to prepare raw data for further analysis by identifying and handling common data quality issues such as missing values, duplicate records, and outliers.

## 📂 Dataset

The dataset used in this project contains demographic and behavioral information. It is used for learning and practicing data cleaning techniques.

> **Note:** The dataset is intended for educational purposes only.

## 🛠️ Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔍 Data Cleaning Process

The following steps were performed:

- Data Understanding
- Checking Data Types
- Handling Missing Values
- Removing Duplicate Records
- Detecting and Handling Outliers using the IQR Method
- Data Validation

## 📊 Output

The cleaned dataset is ready for Exploratory Data Analysis (EDA) and machine learning.
